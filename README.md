This is a simple, client-side web application designed to manage supermarket inventory using **vanilla HTML, CSS, and JavaScript**.

It serves as a basic prototype for a front-end inventory tracking tool, allowing users to track items, monitor stock levels, and identify products that are nearing their expiration date.

## âœ¨ Features

* **View All Items:** Displays a comprehensive list of all products in stock.
* **Low Stock Alerts:** Filters and shows items with a quantity of **less than 5**.
* **Expiration Tracking:** Filters and shows items that are about to expire within the **next 30 days**.
* **Add New Items:** A form to quickly input and add new products to the inventory.
* **Search Functionality:** Filter the inventory by product name or category.
* **Edit/Delete Actions:** Allows for in-line modification or removal of inventory items.
